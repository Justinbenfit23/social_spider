# Social Media Playwright Scraper

## Description:
This project was to become more proficient with the playwright web driver with python. The app does the following:
- Initializes the web driver
- Logs into social media app
- Navigates to each followed account and scrolls through the feed, scraping all posts in the process
- Then parses the date, user handle and post text to file to then run the Roberta NLP pre-trained model and guage sentiment about financial markets
